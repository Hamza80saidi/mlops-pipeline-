# Medical Symptom Classification MLOps Project

A production-ready MLOps pipeline for classifying medical symptoms using machine learning and deep learning models.

## Features

- **Multiple ML Models**: SVM, Random Forest, Logistic Regression, and ALBERT
- **MLOps Stack**: 
  - MLflow for experiment tracking and model registry
  - DVC for data versioning and pipeline management
  - Prometheus & Grafana for monitoring
  - MinIO for S3-compatible object storage
- **API Service**: FastAPI with OAuth2 authentication
- **Containerization**: Docker and Kubernetes deployment
- **Design Patterns**: Strategy, Factory, and Repository patterns
